Features

Everything You Need to Download Faster

Packed with powerful features designed for the best download experience on Android.

Aria2c Speed Boost

Up to 32x parallel connections for blazing-fast downloads. Configurable 2/4/8/16/32+ connection control in Seal Plus Extras.

Gradient Dark Theme

Exclusive glassmorphism UI with deep OLED backgrounds and vibrant blue-purple gradients. Stunning on every screen.

1000+ Platforms

Powered by yt-dlp. Download from YouTube, Instagram, TikTok, Twitter, Facebook, Vimeo, Reddit, and over 1000 other platforms.

Pause & Resume

Full download control with queue support. Pause any download and pick up exactly where you left off — even after app restart.

Auto-Update System

Seamless in-app updates directly from GitHub releases. Always stay on the latest version with one-tap installation.

Custom Commands

Full yt-dlp CLI access with template management. Save and reuse custom commands for advanced power-user workflows.

Open Source

100% open-source under GPL-3.0. No ads, no tracking, no data collection. Full transparency — review every line of code.

Biometric Lock

Protect the app with fingerprint or face unlock. PIN fallback ensures your downloads and settings stay private.

Smart Format Routing

Platform-aware stream-merge logic. YouTube & Reddit use merge path; all other platforms pick the best combined format automatically.

Download Metrics

See real Average Speed (e.g. 3.2 MB/s) and Download Time (e.g. 2m 34s) for every download in the details dialog.

Concurrent Downloads

Run up to 5 simultaneous downloads with smart queue management. Configure the limit from Seal Plus Extras settings.

Advanced Proxy

Built-in proxy support with auto-fetch, auto-test, and custom HTTP/HTTPS/SOCKS configuration. Bypass geo-restrictions with ease.

What's New

Latest release brings smart downloads, connection-level speed control, and real metrics.

⚙️ Aria2c Connection Control

New slider in Seal Plus Extras — choose 2, 4, 8, 16, or 32+ parallel connections to maximize your download speed.

📊 Average Speed & Download Time

Details dialog now shows computed average speed (e.g. 3.2 MB/s) and download duration (e.g. 2m 34s) for every file.

🚀 Smart Stream-Merge Routing

Only YouTube & Reddit use separate stream merging. All other platforms now pick the best combined format directly — faster and smarter.

🎨 Clean Resolution Labels

Format cards now show human-readable titles like "1080×1920" instead of raw yt-dlp format strings for a cleaner selection experience.

🖼️ Redesigned Detail Drawer

Overhauled history drawer with 16:9 thumbnail card, source URL card with copy/open actions, and stats grid layout.

💎 Sponsor Dialog Controls

New schedule picker in Seal Plus Extras — set sponsor dialog frequency to Off, Weekly, or Monthly to suit your preference.

FAQ

Frequently Asked Questions

Download arm64-v8a for most modern phones (2017 and later). If you're unsure, the Universal APK works on every Android device. Choose x86_64 for tablets or emulators with Intel/AMD processors.

Seal Plus automatically checks GitHub releases when you open the app. When a new version is available, a notification appears with the changelog. Tap once to download and install — no manual visits to GitHub needed. You can disable this in Settings → About.

Go to Settings → Look & Feel, enable Dark Theme, then toggle the Gradient Dark switch. The theme activates immediately with vibrant blue-purple glassmorphism effects across the entire app.

Seal Plus supports 1000+ platforms via yt-dlp — including YouTube, Instagram, TikTok, Twitter/X, Facebook, Reddit, Vimeo, Twitch, SoundCloud, Bandcamp, Bilibili, Dailymotion, and many more. View the full list →

Completely safe. Seal Plus is 100% open-source (GPL-3.0), contains no ads, no trackers, and no data collection. All processing happens locally on your device. You can inspect every line of code on GitHub.

Seal Plus is an enhanced fork of the original Seal app by JunkFood02, adding: exclusive Gradient Dark Theme, built-in auto-update system, advanced aria2c with configurable connections, download metrics (speed + time), smart format routing, and active maintenance by Mahesh Technicals.

Try these steps: (1) Update yt-dlp via Settings → About → Update yt-dlp. (2) Check your internet connection. (3) Use the Retry button on the failed download card. (4) For geo-restricted content, configure a proxy in Settings → Network. (5) Report persistent issues on GitHub Issues.
